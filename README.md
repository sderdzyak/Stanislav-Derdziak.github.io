# Stanislav Derdziak — Personal CV Website

A simple, responsive personal CV and portfolio website built with pure HTML and CSS. It serves as an online résumé presenting my background, experience, and certifications as a Helpdesk Engineer.

## 📖 About the Project

This is a multi-page landing-style website hosted on GitHub Pages. The homepage opens with an article on the role of a Helpdesk Engineer and why human expertise still matters in an AI-driven world. From there, visitors can navigate through dedicated pages covering my background, work experience, certifications, and contact details.

## ✨ Features

- **Fully responsive layout** — adapts cleanly to desktop, tablet, and mobile screens
- **Multi-page structure** with shared navigation across all sections
- **Pure HTML & CSS** — no frameworks, no JavaScript, no build step
- **Hosted on GitHub Pages** for free, fast static delivery
- **Links to external profiles** — LinkedIn, GitHub, and HackerRank

## 📂 Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — intro and article on the Helpdesk Engineer role |
| `About.html` | About me — background and personal summary |
| `Experience.html` | Work experience and skills |
| `Badge.html` | Certifications and earned badges |
| `Contact.html` | Contact information and links |

## 🛠️ Built With

- **HTML5** — semantic markup and page structure
- **CSS3** — styling, layout, and responsive design (separate stylesheet per page)

## 🚀 Running Locally or via Live demo:

Clone the repo and open `index.html` in any modern browser — no installation or build tools required:

```bash
git clone https://github.com/sderdzyak/Stanislav-Derdziak.github.io.git
cd Stanislav-Derdziak.github.io
```

Then double-click `index.html` or open it in your browser.
